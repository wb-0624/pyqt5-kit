# 项目说明

为了快速搭建一个pyqt5的项目，写了这个工程。

# 启动

运行前，要把运行目录改为工程根目录。

![img.png](assets/doc/run_dir.png)

# 各文件夹的作用

## animation

PyQt动画相关类，用于直接生成动画效果，还在完善中。

## app_config

应用配置。包括应用常量 和 图标常量。

## assets

资源文件。

- doc 文档相关图片。
- font 字体文件。
- gif 动图。
- img 图片。

## doc

文档。

## example

示例。
run `example/demo.py`

## script

自己写的一些脚本，用于快速生成一些文件。

## theme

主题相关。

## utils

工具类。

## widget

自定义控件。
